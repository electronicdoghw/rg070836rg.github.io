夏令营

发表于 2019-03-14 | 分类于 夏令营

#苏州大学 计算机学院夏令营
苏大夏令营一共是2天，连上前一天的报告，总共2天半的时间。

报道：
苏大计算机，在相门地铁站下车即到达，所以可以坐火车到苏州站，转地铁，大约20分钟不到，就可以到达苏大。这次夏令营是苏大计算机学院第一次举办，我们去的时候，安排的是住的校外的宾馆。去教学楼报道，领取牌子，下午入住宾馆，第一天的行程就结束了。然后我们晚上去平江路玩了玩。

活动第一天
上午：7个小组，每组15分钟左右，宣讲各组做的事情，大约一共俩小时，随便听听就好。
中午：在北疆饭店吃饭，回宾馆休息一会
下午：自由去每个小组实验室参观，苏大计算机学院还是很大的，可以说有4栋楼，2栋主楼，2栋附楼。我去了“中文信息处理和自然语言理解”组还有“先进数据分析”组，这边随意过去，听介绍，这俩组实力比较强，值的过来。其他组，感兴趣就可以看看，没有任何的限制。
晚上：做PPT，第二天面试需要有个人展示环节

活动第二天
上午：面试，一共7组，每组12个人。有些组是单独进去面试，我们组，是所有人坐在一起，可以互相看到其他人的表现。面试主要由个人陈述为主，五分钟，注意详略得当，我感觉我就说了一会会就超时了。然后老师，会根据你的PPT和表述，进行提问。PPT里面，主要提到了项目经验。这边注意，不要给自己挖坑，例如我们组有位同学，把他的成绩单拉了上来，说他XX课考的非常高，然后老师就开始问这门课的一些东西，然而回答的并不是很好，所以，除非你准备的非常充分，否则，不要自行提出，对某门课非常熟悉，非常擅长这种话，老师总会问道你不会的地方。总体来说，以表现自己为主，有些人没有项目经验，或者项目很少的时候，老师就会问，你这个学科和其他学科的区别/苏州美么?/为什么来苏大这种问题，所以，有项目经验，就比较占优势。

下午：颁发优秀营员/闭营
苏大严格控制夏令营优秀营员为40%，所以每组只能有5个优秀营员，还是比较残酷的，这时候，面试表现就显得十分重要，闭营时间很快，前后大约40分钟，大概3点之前，整个夏令营就结束了，就可以收拾东西走人了。

总结：苏大计算机，nlp组和ADA组做的不错，老师也都很好，有兴趣的小伙伴可以来试试。

发表于 2019-03-14 | 分类于 夏令营

暑假去的夏令营，是华师大数据所夏令营，很不错的一个学校，夏令营中，有一个半天是上机测试，语言任选，改题是由实验室师兄师姐，过来输测试用例，通过一题打钩，相比隔壁软件夏令营，这边的改题，还是很人性化的，软件夏令营用的是在线OJ，据说题目有点难。下面是题目，以及当时做的时候，遇到的一些问题和大概的想法。

#A 高精度乘法

这一题是经典题，思路是模拟运算，模拟我们笔算的过程，本来应该是不难的，但是在写的过程中，老不对。浪费了半小时之后，放弃了，后直接用Java中BigInteger，几行代码就过了。

#B 判断三角形类型

简单题，用余弦定理，几行代码搞定。关键就是瞬间想不起来余弦定理，推导了好久。

#C 电梯问题

水题，简单的模拟，看懂题目，手算步骤出来之后，直接码代码。

#D 回文素数

水题，会写判断回文的函数，这条就搞定，现场的测试用例，在500以内。

#E 寻找字符串

主要考字符串的一些算法，这题我一开始没写，后来发现很好写，但是时间不够了，这题写了一半，主要思路：在和学姐确认，所有测试用例，只含有一对[]之后，调用string中的find函数，找到左右括号的位置，读取之间的字符，扩充相应的倍数，然后递归调用find查找出现次数即可。

#F 寻找最大值

动态规划基本题，学过算法的应该都做过，不是很难。

#G

搜索，也是很经典的题目，肯定都做过。

#H

八数码，宽搜，并没有写过，直接放弃了

#总结
总之，最后交了6题，还有一题没写完，一题没看，华师大的题目还是很基础的，希望想去华师大数据所的的同学，可以来看看。

// LeetCode, Permutations
// 深搜，增量构造法
// 时间复杂度 O(n!)，空间复杂度 O(n)
class Solution {
public:
vector<vector > permute(vector& num) {
sort(num.begin(), num.end());
vector<vector> result;
vector path; // 中间结果
dfs(num, path, result);
return result;
}
private:
void dfs(const vector& num, vector &path,
vector<vector > &result) {
if (path.size() == num.size()) { // 收敛条件
result.push_back(path);
return;
}

// 扩展状态
for (auto i : num) {
// 查找 i 是否在 path 中出现过
auto pos = find(path.begin(), path.end(), i);
if (pos == path.end()) {
path.push_back(i);
dfs(num, path, result);
path.pop_back();
}
}
}
};
