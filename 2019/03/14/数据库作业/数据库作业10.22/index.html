1.SQL语言是（B）的语言，容易学习 。

A. 过程化
B.非过程化
C.格式化
D.导航式

2.在视图上不能完成的操作是（C） 。

A.更新视图
B.查询
C.在视图上定义新的表
D.在视图上定义新的视图

3.SQL语言集数据查询、数据操纵、数据定义和数据控制功能于一体，其中，CREATE、DROP、ALTER语句是实现哪种功能（C）。

A.数据查询
B.数据操纵
C.数据定义
D.数据控制

A.CREATE TABLE
B.DROP VIEW
C.CREATE VIEW
D.GRANT

A.DELETE TABLE S
B.DELETE S
C.DROP TABLE S
D.DROP S

A. ADD TABLE S（CN CHAR（８））
B. ADD TABLE S ALTER（CN CHAR（８））

C.UPDATE S Age
D.ALTER TABLE S ‘Age’
